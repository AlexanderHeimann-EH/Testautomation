PTVS remote debugging proxy for Microsoft Azure Web Sites

Python remote debugging is enabled for this Microsoft Azure web site, and it can be attached to using Python Tools for Visual Studio. To attach, go to Debug → Attach to Process, switch Transport to "Python remote debugging", and enter the following URL in the Qualifier textbox:

{{WS_URI}}

where secret is the value of WSGI_PTVSD_SECRET in web.debug.config of the project from which this web site was deployed.
